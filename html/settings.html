Đồng bộ Google Sheets

Cấu hình API Web App Google Apps Script và mã lực bảo mật kết nối.

Kết nối API

Vui lòng nhập đường dẫn URL hợp lệ.
Vui lòng nhập mã Access Token.

Hướng dẫn Apps Script

1. Mở Google Sheet, chọn Extensions > Apps Script.

2. Dán đoạn mã sau vào dự án:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  
  if (data.accessToken !== "MÃ_ACCESS_TOKEN_CỦA_BẠN") {
    return ContentService.createTextOutput("Access Denied").setMimeType(ContentService.MimeType.TEXT);
  }
  
  sheet.appendRow([
    data.timestamp, data.doanhChuName, data.e1, data.e2, data.e3, data.motivation,
    data.a2, data.totalMembers, data.distributionRatio, data.culture, data.leadershipStyle,
    data.d1, data.d2, data.skills.join(", "), data.a1.join(", "), data.lvpStyle,
    data.lvpIntra.join(", "), data.lvpInter.join(", "), data.lvpBrand, data.b1, data.b2,
    data.c1.join(", "), data.c2
  ]);
  return ContentService.createTextOutput("Success").setMimeType(ContentService.MimeType.TEXT);
}

3. Chọn Deploy > New Deployment, chọn loại Web App.

4. Thiết lập Execute as: Me và Who has access: Anyone, sao chép URL dán vào ô bên cạnh.
